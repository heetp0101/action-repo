# GitHub Actions Webhook Testing Repository

This is a sample repository for testing GitHub webhooks integration with MongoDB.

## Quick Start

See README.md for complete setup and webhook configuration instructions.

## File Structure

This is a simple repository designed to trigger webhook events. Add your project files here.

## Testing Events

1. **PUSH Event** - Make a commit and push
2. **PULL_REQUEST Event** - Create a PR from a new branch
3. **MERGE Event** - Merge the PR back to main

Refer to README.md for detailed instructions.
